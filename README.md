# 🔥 Multi-Task Deep Learning System for Smart Surveillance

## 📌 Overview
This project presents a multi-task deep learning framework for real-time surveillance, integrating:

- 🔥 Fire Detection  
- 🚶 Fall Detection  
- ✋ Gesture Recognition  

The system is designed for intelligent monitoring environments such as smart homes, elderly care, and security systems.

## 🚀 Key Features
- Multi-task learning architecture  
- Transfer Learning using MobileNetV2  
- LSTM-based gesture recognition using hand landmarks  
- Real-time detection capability  
- Lightweight and scalable pipeline  

## 📊 Results
| Task | Accuracy |
|------|--------|
| Fire Detection | 95.65% |
| Fall Detection | 97.00% |
| Gesture Recognition | ~90% |
